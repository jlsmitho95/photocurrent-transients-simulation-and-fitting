This repository contains three R codes corresponding to: 

1. Photocurrent_Diffusion_submit.r : Allows to simulate the photocurrents according to theoretical values provided for the kinetic constants and the concentrations of the supporting electrolyte, electron donor and acceptor.
2. Photocurrent_NLF_0.85V-k3468-examples: Generates a matrix for the initial parameters to estimate using a Latin Hypercube followed by the non-linear regression to the experimental data. This is for k3, k4, k6 and k8 only. 
3. Photocurrent_NLF_0.85V-k3468-examples: Generates a matrix for the initial parameters to estimate using a Latin Hypercube followed by the non-linear regression to the experimental data. This is for k3, k4, k6 and k8 only.
4. Data_corrected-12-02-2020-3-1.94-5-DP-1.csv: Corresponds to the raw experimental data for the experiment of photoexcitation of ZnTPPC at the liquid|liquid interfaces.
